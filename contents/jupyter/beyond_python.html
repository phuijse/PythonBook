8. Otros lenguajes en Jupyter¶

A lo largo de este libro se trabaja principalmente con Python y con el kernel de IPython para jupyter

Sin embargo existe una lista de kernels desarrollados por la comunidad para otros lenguajes

  • IRkernel

  • IJulia

  • Ijavascript

  • IHaskell

  • IRuby

  • IElixir

  • Xeus Cling (C++)

Nota

Algunos de estos se pueden probar en jupyter.org/try

También hay lenguajes que tienen magias oficiales para usarse dentro del kernel de Python. Estos requieren que sus ejecutables respectivos estén disponibles

Por ejemplo

%%bash

echo "Hello, world!" 

Hello, world!

%%perl
print "Hello, World!\n";

Hello, World!

%%ruby

puts 'Hello, world!'

Hello, world!
